
# Website - HTMLs

Bootstrap 5 Boilerplate, Es6 with Gulp 4+. Sass, browser-sync.

Install $ `npm install` <br/>
Init $  `npm init` <br/>
Start $ `gulp serve` <br/>
Prod $ `npm run prod` <br/>
